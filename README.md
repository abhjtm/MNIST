# MNIST
## 1. Plain vanilla model
### Model features
* Number of parameters: 14,322
* 1 MaxPooling layer
* No batch normalization, no dropout

### Results
#### After 40 epochs:
* validation accuracy peaked to 98.9%
* training accuracy hovered around 99.4%

#### After 80 epochs:
* validation accuracy remained at ~99%
